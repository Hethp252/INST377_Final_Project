# INST377_Final_Project

# Description:
This application connects directly to the Polygon API for the current market data and allows users to save target watchlists which they can access in the future through Supabase.

## Target browsers: 
Any browser can access this website. For example: It can be used on Android and IOS on smartphones if a user is outside where they can't access their computer and wants to check their portfolio. They can also just use it on any computer whether it is on Windows, Mac or Linux as long as a device has a browser it can be used and accessed.
